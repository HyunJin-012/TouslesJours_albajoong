# TouslesJours_albajoong

## 게임과 설명 영상에 앞서서...
#### 코로나 학번 정읍읍과 이읍읍은 중앙대학교 뚜레쥬르 알바 공고를 확인하고 지원하게 된다. 
#### 17:1의 경쟁률을 뚫고 정읍읍과 이읍읍은 2차 줌 면접을 보게 된다.
#### 알바가 간절했던 그들은 꼰대 점장을 만나 줌 면접으로 빠르고 정확한 계산 능력을 테스트받게 되는데...
#### 게임설명 영상: https://www.youtube.com/watch?v=gjhKiUGYaKw

## 게임 설명
#### 1. easy, normal, hard 3가지의 모드가 있다.
#### - easy모드에서 당신은 주문당 20초 이내에 계산을 마쳐야 하며 총 10번 주문을 받게 된다. 이 때 최대 제품 개수는 3개이다.
#### - normal모드에서 당신은 주문당 15초 이내에 계산을 마쳐야 하며 총 20번 주문을 받게 된다. 이 때 최대 제품 개수는 4개이다.
#### - hard모드에서 당신은 주문당 12초 이내에 계산을 마쳐야 하며 총 30번 주문을 받게 된다. 이 때 최대 제품 개수는 5개이다.

#### 4. 포스기 활용 방법
#### - 당신은 포스기의 100원, 500원, 1000원, 5000원 버튼을 눌러 거스름돈을 손님에게 주어야 한다.
#### - 당신이 done버튼을 누르면 현재까지 버튼을 눌러 획득한 만큼의 돈을 거스름돈으로 제공한다.
#### - 당신이 reset버튼을 누르면 현재까지 버튼을 눌러 획득한 돈이 0원으로 초기화 된다.
